## 1. 系統架構總覽

本系統分為**前台（顧客點餐）**與**後台（管理員管理）**兩大部分，共用同一套 Vue 3 應用程式。

```
前台路由（顧客使用）
├── /              首頁
├── /menu          菜單列表
├── /product/:id   商品詳情
├── /cart          購物車 & 結帳
└── /order/:id     訂單確認 & 付款

後台路由（管理員使用，需登入）
├── /#/admin              登入頁
├── /#/admin/products     產品管理
├── /#/admin/orders       訂單管理
├── /#/admin/coupons      優惠券管理
├── /#/admin/banners      輪播圖管理
├── /#/admin/features     特色區塊
├── /#/admin/cta          CTA 區塊
└── /#/admin/footer       Footer 編輯
```

## 2. 前台功能說明

### 2-1. 首頁

| 區塊 | 說明 |
|------|------|
| **Hero 輪播** | Swiper 自動輪播，每 4.5 秒切換，支援左右箭頭與圓點分頁 |
| **特色區塊** | 三欄圖片卡片，滑鼠 Hover 展開說明文案 |
| **CTA 區塊** | 全背景圖片，含主標題、描述與行動按鈕 |
| **Footer** | 餐廳資訊、社群媒體圖示、Google 地圖、版權 |

### 2-2. 菜單頁（`/menu`）

- **分類篩選**：點選分類標籤（壽司、拉麵、丼飯⋯⋯）快速過濾
- **關鍵字搜尋**：輸入料理名稱即時篩選
- **加入購物車**：點擊「加入」按鈕，右上角購物車徽章即時更新
- 點擊商品圖片或名稱可進入商品詳情頁

### 2-3. 商品詳情頁（`/product/:id`）

- 顯示大圖、分類標籤、描述、售價（含原價劃線）
- 數量選擇（＋ / －）
- 加入購物車後顯示 Toast 通知

### 2-4. 購物車 & 結帳（`/cart`）

**用餐方式選擇（切換選項）：**

| 模式 | 需填資料 |
|------|---------|
| 🍽 **內用** | 桌號（必填）、備註（選填） |
| 🛍 **外帶** | 姓名、電話、Email、地址（必填）、備註（選填） |

**優惠券：**
在購物車頁輸入優惠碼 → 點「套用」即可折扣。

| 優惠碼 | 折扣 |
|--------|------|
| `DISCOUNT80` | 全單 8 折 |
| `BIRTHDAY10` | 全單 9 折 |
| `SAVE1000` | 全單 9 折 |
| `SAVE2000` | 全單 9 折 |

**下單流程：**
填好資料 → 點「確認下單」→ 跳轉至訂單確認頁

### 2-5. 訂單確認 & 付款（`/order/:id`）

- 顯示訂購商品明細、聯絡資料、總金額
- 點「立即付款」完成付款，按鈕轉為「已完成付款」綠色狀態

## 11. 技術架構

| 類別 | 技術 |
|------|------|
| 前端框架 | Vue 3（Composition API） |
| 狀態管理 | Pinia + pinia-plugin-persistedstate |

### 資料儲存說明

| 資料類型 | 儲存位置 |
|---------|---------|
| 產品、訂單、優惠券 | 遠端 API（HexSchool 伺服器） |
| 輪播圖、特色區塊、CTA、Footer | 瀏覽器 localStorage（Pinia persist） |
| 登入 Token | Cookie（含到期時間） |

> ⚠️ localStorage 資料儲存於**各自瀏覽器**，不同電腦或清除快取後需重新設定。

## 12. 部署說明

### 自動部署

每次推送至 GitHub `main` 分支，**GitHub Actions 自動執行**：
1. `npm ci` 安裝依賴
2. `npm run build` 建置 `dist/` 目錄
3. 上傳至 GitHub Pages

### 手動觸發部署

前往 GitHub → Actions → Deploy to GitHub Pages → **Run workflow**

### 本機開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器（預設 http://localhost:5173）
npm run dev

# 建置生產版本
npm run build
```

### 新增產品腳本

```bash
# 批次新增日式料理產品（28 項）
node --input-type=module < scripts/seed-products.js

# 批次新增優惠券（4 張）
node --input-type=module < scripts/seed-coupons.js
```
